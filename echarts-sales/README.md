# echarts-sales

## 项目介绍

大数据可视化的应用

## 技术栈

前端：vue3、vue-cli5.x、vue-router3.x、Echarts5.0、axios、flex 弹性布局
后端：node.js express
插件：less、lib-flexible(rem 屏幕适配)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
